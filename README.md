# Loenbro Mission Critical Project Map

Interactive US map of Loenbro locations and mission-critical projects.

**Live site:** https://blancomat9-dev.github.io/Loenbro-map/

- Single self-contained HTML file (`index.html`) - no build step, no external dependencies.
- All project data is fictional demo data by design.
- Data can be refreshed per-session in the browser via the Import Excel button; the default dataset is baked into the file.

To update the live site, replace `index.html` and push (or upload via the GitHub web UI). GitHub Pages redeploys automatically in about a minute.
